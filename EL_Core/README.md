# EL Core

This folder contains core empirical likelihood (EL) utilities and estimator
functions used in the manuscript:

**Empirical Likelihood in Causal Inference Settings**

Files:
- `el_core.R`: core EL solvers, weights, and EL-weighted kernel density utilities
- `estimators.R`: EL and Normal stabilized-weight estimators for continuous GPS

Application datasets are restricted and not included.
